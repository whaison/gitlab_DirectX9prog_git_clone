このテキストファイルは、付属CD-ROM「Disk2」について紹介しています。

■収録内容について
本CD-ROMはWindows用です。

本誌で解説している、Visual Studio.NET (2002)用のプロジェクト用のファイルと、ソース
コード、ならびに実行形式のファイル（.exe）を収録してます。

　各フォルダ名の先頭の数値は、本誌のChapter番号と同じになっています。
　「main.exe」は、各節で解説しているアプリケーションプログラムとなっています。

　収録されているサンプルコードをビルドさせるには、Visual C++ .NET、Visual Studio
 .NET などの開発環境が必要です。
　本書付属のCD-ROMには、これらの開発ツールは収録されておりません。

※収録されているプロジェクト用のファイルを「Visual Studio.NET 2003」で読み込むと、
形式が古いなどのダイアログが出ることがあります。これらはそれらのダイアログに従って
お使いのバージョンに変換してから、利用するようにしてください。
（収録のVisualStudioProjectのProjectTypeは"Visual C++"、バージョンは"7.00"、
　ソリューションファイル（Solution File）の Format Version は 7.00　です）

　開発の際には、CD-ROMから各プロジェクトのフォルダごと、ハードディスク領域などに
コピーして利用してください。Windows XP以外のOSでは、コピーしたファイルに読み取り
専用属性が付くことありますので、これらを解除してから利用してください。

　「main.exe」を他のマシンで実行する際には、同じフォルダに収録されているテクスチャ
ファイル（.bmp）やXファイル（.x）など関連のファイルも同時に必要となります。
